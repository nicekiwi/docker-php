# docker-php

## PHP-FPM 8.x Docker Image

Optimised for Laravel 9.

- Alpine 3.x
- PHP 8.x
- Composer 2.x

Supports Redis, MySQL and PostgreSQL on `amd64` and `arm64/v8` architectures.

## Build ENV variables

```
# Defaults
PHP_VERSION=8.2.2
ALPINE_VERSION=3.17
WORKDIR=/var/www
TZ=UTC
```
